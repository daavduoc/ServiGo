

## SERVIGO

## 📌 Descripción

SERVIGO es una plataforma web tipo marketplace de servicios, diseñada para permitir a los usuarios contratar y agendar servicios profesionales y técnicos, ya sea a domicilio o en un lugar establecido.

## 📍 Funcionalidades principales

- 🔎 Búsqueda inteligente basada en geolocalización
- 📅 Visualización de disponibilidad en tiempo real
- 👤 Perfiles detallados de prestadores
- 🔒 Sistema de seguridad con reconocimiento facial

## 🎯 Objetivo

Facilitar la búsqueda rápida, segura y eficiente de servicios cercanos, tanto en situaciones urgentes como en el uso cotidiano.

## 🧠 Valor agregado

- Servicios ordenados por cercanía
- Información clara sobre disponibilidad (Disponible / En atención / Cerrado)
- Validación de identidad para mayor seguridad

 🔐 Seguridad

La plataforma incorpora un sistema de reconocimiento facial para validar la identidad de prestadores y usuarios en servicios a domicilio.
